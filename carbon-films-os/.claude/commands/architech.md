Execute o Architect Agent da Carbon Films para a seguinte entrada: "$ARGUMENTS"

Passos:
1. cd /root/carbon-films-os
2. Execute: python commands/architect.py "$ARGUMENTS"
3. Mostre o output completo ao usuário

Se ele criar um objetivo para o Manager, informe o usuário que pode usar /manager para processar.
