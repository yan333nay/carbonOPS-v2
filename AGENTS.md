# Carbon OPS v2 — AGENTS.md

## Contexto do Projeto
Sistema de agentes de IA da Carbon Films. 8 Clones de IA (CEO-ATLAS, NOVA, VECTOR, MIRA, FINN, PULSE, FLUX, SAGE) orquestrando vendas, operações, financeiro e conteúdo da agência.

## PROTOCOLO DE INÍCIO DE SESSÃO OBRIGATÓRIO

Quando o usuário disser "oi" ou qualquer saudação ao iniciar uma sessão nesta pasta:

1. **Consultar o Notion** — Ler o Checklist de Implementação (página ID: `34859fc2-d3b9-80f6-8ee6-d132e0ef5c7a`) para ver o estado atual de cada item
2. **Consultar o GitHub** — Verificar commits recentes em `https://github.com/yan333nay/carbonOPS-v2` para capturar atualizações
3. **Ler o arquivo de progresso local** — `progresso.md` nesta pasta
4. **Reportar ao usuário:**
   - Em qual fase/item estamos
   - O que foi concluído desde a última sessão (baseado no GitHub e Notion)
   - O próximo item a executar
   - Qualquer bloqueante pendente

Não espere o usuário perguntar. Faça isso automaticamente ao primeiro "oi".
