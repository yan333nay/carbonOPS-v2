# Task: scheduler-send-alarm

**Agent:** Rhythm (scheduler)
**Command:** `*send-alarm`
**Trigger:** Cron schedule — 5 times daily (08:00, 12:00, 17:00, 19:00, 22:00 BRT)

---

## Purpose

Send a contextual WhatsApp alarm message at each scheduled time. Each alarm has a specific purpose and includes real-time data from the goals tracker.

## Alarm Schedule

| Time | Purpose | Data Required |
|------|---------|--------------|
| 08:00 | Morning checklist + daily plan | Weekly progress, today's pending |
| 12:00 | Stories afternoon window reminder | Stories posted today |
| 17:00 | Feed prime time reminder | Feed posts today |
| 19:00 | Reels/TikTok prime time | Reels/TikTok count this week |
| 22:00 | End-of-day summary | Published today, tomorrow's tip |

## Execution Steps

```
1. Load goals tracker (data/goals-tracker.yaml)
2. Calculate current progress vs targets
3. Identify which alarm template to use (by time)
4. Fill template variables with real data
5. Send via WhatsApp integration
6. Log alarm sent in session log
```

## 08:00 — Morning Checklist

Data to collect:
- `feed_published` / `feed_target` (this week)
- `stories_published` / `stories_target` (this week)
- `reels_published` / `reels_target` (this week)
- Today's suggested content based on remaining targets + day of week

Message:
```
Bom dia! ☀️ Semana {week_number} — dia {day_of_week}

📊 Progresso:
• Feed: {feed_published}/{feed_target} posts
• Stories: {stories_published}/{stories_target}
• Reels/TikTok: {reels_published}/{reels_target}

🎯 Sugestão de hoje:
{daily_suggestion}

Envie foto + /comando para começar.
```

Daily suggestion logic:
- Monday: `/autoridade` or `/carrossel-edu` (educational start of week)
- Tuesday: `/feed-impactante` or `/resultado-cliente` (engagement day)
- Wednesday: `/reels-trend` or `/tiktok-viral` (mid-week reach push)
- Thursday: `/carrossel-edu` or `/autoridade` (educational content)
- Friday: `/bastidores` or `/reels-trend` (end-of-week personal)
- Saturday: `/bastidores` (weekend casual)
- Sunday: `/resultado-cliente` (reflection + proof)

## 12:00 — Stories Window

Data to collect:
- `stories_today` (stories published today)
- `stories_remaining_today` (3 - stories_today)

Message:
```
⏰ Janela de stories da tarde (12h-13h)

{
  if stories_today == 0: "Nenhum story hoje ainda. Hora de começar!"
  if stories_today == 1: "1 story publicado. Faltam 2 para a meta do dia."
  if stories_today == 2: "2 stories publicados. Falta 1 para fechar o dia."
  if stories_today >= 3: "Meta do dia batida! Stories completos ✅"
}

Envie foto + /stories-simples
```

## 17:00 — Feed Prime Time

Data to collect:
- `feed_today` (feed posts today)

Message:
```
📸 Hora nobre do Instagram (17h-18h)

{
  if feed_today == 0: "Nenhum post de feed hoje. Esta é a melhor janela do dia."
  else: "Post do dia: {feed_today_title}. Quer postar outro?"
}

Engajamento médio às 17h é 31% maior que outros horários.

/feed-impactante | /autoridade | /resultado-cliente
```

## 19:00 — Reels/TikTok Prime Time

Data to collect:
- `reels_this_week` / `reels_target`

Message:
```
🎬 Horário nobre Reels e TikTok (19h-20h)

Meta semanal de vídeos: {reels_this_week}/{reels_target}

{
  if reels_this_week < reels_target:
    "Faltam {remaining} vídeos para a meta. Este é o melhor horário para alcance orgânico."
  else:
    "Meta batida! Postar agora aumenta alcance em até 40% vs outros horários."
}

/reels-trend | /tiktok-viral
```

## 22:00 — End-of-Day Summary

Data to collect:
- All posts published today
- Pending items
- Tomorrow's best tip (from mind-council)

Message:
```
📊 Resumo do dia — {date}

✅ Publicado:
{published_list or "Nenhum post hoje"}

{
  if any pending:
    "📋 Pendente: {pending_list}"
}

💡 Dica de amanhã ({tomorrow_day}):
{tomorrow_tip}

Boa noite! 🌙
```

## Tomorrow's Tip Source

Generated by Kiso mindclone based on:
- Current week's performance (what's working)
- Tomorrow's day of week (best format)
- Trends data from Pulse (trend-analyst)

```yaml
  data_snapshot:

    reels_progress: 'X/Y'
```
