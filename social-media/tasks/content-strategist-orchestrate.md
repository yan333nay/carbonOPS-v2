# orchestrate

Orchestrate the full squad execution for the current week, spawning specialized agents for each content production step.

## Orchestration Flow

```
1. Intelligence Gathering
   → Task(trend-analyst, *scan-trends)
   → Task(trend-analyst, *suggest-content-from-trends)
   → Task(performance-analyst, *collect-metrics) [last week]

2. Video Content Production (parallel per video post)
   → Task(video-specialist, *create-video-brief) [for each Reel]
   → Task(video-specialist, *suggest-trending-audio) [for each Reel]

3. Copy Production (parallel per post)
   → Task(copy-writer, *write-caption) [for each post]
   → Task(copy-writer, *create-hook) [for each post]
   → Task(copy-writer, *generate-hashtags) [for each post]

4. Scheduling
   → Task(scheduler, *generate-content-calendar) [full week]
   → Task(scheduler, *optimize-posting-times) [if new platform data]

5. Quality Review
   → Review all content against content_pillars
   → Flag any off-brand content for revision
   → Compile weekly_content_package
```
