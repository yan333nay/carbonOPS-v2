# Daily Post Checklist

Use this checklist every day before, during, and after publishing. Run via `*daily-checklist` command.

---

## PRE-PUBLISH (Before 9am)

### Content Verification
- [ ] All posts scheduled for today are confirmed in the scheduling tool
- [ ] Media assets (images/videos) are attached and correct resolution
- [ ] Caption is final (no "DRAFT" or placeholder text)
- [ ] Hashtags are included and relevant
- [ ] CTA is present and clear
- [ ] Brand handle or tag is correct (no typos)
- [ ] Links in bio are correct and working (if post references link in bio)

### Quality Check
- [ ] Caption has no spelling errors
- [ ] Hook (first line) is strong — would YOU stop scrolling?
- [ ] Emojis used appropriately (if applicable)
- [ ] No competitor handles tagged accidentally

### Video/Reel Specific
- [ ] Video is 9:16 aspect ratio (Reels/TikTok)
- [ ] No text is cut off at edges
- [ ] Audio is properly set (not muted)
- [ ] Subtitles/captions added (accessibility)
- [ ] Cover/thumbnail is eye-catching

---

## PUBLISH (During scheduled time)

- [ ] Post went live at scheduled time
- [ ] Quickly verify post looks correct on mobile view
- [ ] Pin comment if needed (additional context/link)

---

## POST-PUBLISH (Within 1h of publishing)

### Engagement Activation
- [ ] Reply to first comments within 30 minutes (boosts algorithm)
- [ ] Like comments to encourage more
- [ ] Share to Stories if post performs well (optional)

### Monitor
- [ ] Check initial reach and impressions after 1h
- [ ] Note anything unusual (great or poor performance)

---

## END OF DAY (After 6pm)

- [ ] Screenshot metrics for each post published today
- [ ] Log metrics in weekly tracking sheet
- [ ] Respond to any remaining comments
- [ ] Reply to all DMs received
- [ ] Confirm tomorrow's content is ready to go
- [ ] Flag any issues for morning standup
