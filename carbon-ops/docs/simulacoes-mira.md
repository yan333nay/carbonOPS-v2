**Critério de aprovação:** 8/10 respostas aprovadas sem ajustes significativos

## MÓDULO A — REVISÃO EDITORIAL DE SAGE

### CENÁRIO A01 — Draft Bom (deve aprovar com ajustes menores)

**Input para MIRA:**
Draft enviado por SAGE:
```
SAGE DRAFT — Caption Instagram — Restaurante Trovatore — 22/04/2026
Canal: Instagram Feed
Objetivo: Engajamento + autoridade
Hipótese de tom: "casual e apaixonado pela gastronomia"

---
Às vezes o que você precisa não é de uma receita nova.
É de um lugar que lembra por que você gosta de comer.

No Trovatore, cada prato tem história — e a gente conta enquanto você saboreia.

Hoje tem [prato do dia] esperando por você. Reserve sua mesa pelo link na bio.

#trovatore #gastronomia #florianopolis #restauranteFlorianopolis
---

Notas para MIRA:
- Assumi tom nostálgico/emocional. Se o Trovatore tem posicionamento mais moderno, ajustar.
```

**Critérios de aprovação da revisão:**
- Aprova o draft com ajustes menores (não rejeição)
- Identifica o ponto forte: linha 1 tem gancho emocional sem clichê
- Identifica o ponto fraco: "cada prato tem história" é genérico — pede especificidade
- Sugere melhorias concretas, não vagas ("troque X por Y", não "melhore o texto")
- Hashtags: aprova a maioria, pode sugerir 1-2 mais segmentadas
- Mantém o tom escolhido por SAGE — não reescreve do zero

**Resposta ideal (referência):**
```
REVISÃO MIRA — Trovatore — 22/04/2026
Status: REVISÃO NECESSÁRIA (pequena)

Ajustar:
1. "cada prato tem história" → muito genérico. Troque por algo específico do Trovatore:
   "o nhoque da nonna que abriu a casa lá em 2003" ou o que for real para eles.
   SAGE: perguntar ao cliente qual prato tem história mais icônica.

2. CTA: "Reserve sua mesa pelo link na bio" → funciona, mas pode testar
   "Reserva pelo link — últimas mesas do jantar de sexta." (urgência real, se verdadeira)

Aprovado para publicação em: [após ajuste do item 1]
```

**Red flag que MIRA NÃO deve fazer:** reescrever todo o draft, rejeitar por ser "bom mas pode melhorar".

### CENÁRIO A02 — Draft Fraco (deve barrar com direção clara)

**Input para MIRA:**
Draft enviado por SAGE:
```
SAGE DRAFT — Caption — Clínica OdontoMax — 22/04/2026
Canal: Instagram Feed
Objetivo: Conversão (agendar consulta)

---
Você sabia que sorrir é o melhor remédio? 😁

Na OdontoMax temos os melhores profissionais e a tecnologia mais avançada para cuidar do
seu sorriso! Venha nos visitar e descubra como podemos transformar a sua vida!

Agende sua consulta: [telefone]

#dentista #odontologia #sorriso #saude #joinville
---
```

**Critérios de aprovação:**
- MIRA barra o draft claramente
- Identifica os 3 problemas: título fraco (clichê), jargão genérico ("melhores profissionais",
  "tecnologia mais avançada"), CTA sem contexto
- Fornece direção específica para SAGE reescrever (não reescreve ela mesma)
- Tom: orientador, não crítico pessoal
- Indica qual informação específica SAGE precisa coletar do cliente para melhorar

**Resposta esperada:**
```
REVISÃO MIRA — OdontoMax — 22/04/2026
Status: REVISÃO NECESSÁRIA (significativa)

Problemas:
1. Gancho fraco: "Você sabia que sorrir é o melhor remédio?" é o clichê mais batido
   da odontologia. Nenhum paciente para o scroll com isso.

2. Genérico: "melhores profissionais" e "tecnologia avançada" — toda clínica diz isso.
   Qual é o diferencial real da OdontoMax? Especialidade, equipamento específico,
   localização, tempo de atendimento? SAGE: perguntar ao cliente o que os pacientes
   mais comentam nas avaliações deles.

3. CTA: "Venha nos visitar" e "agende" sem contexto. O que acontece quando agenda?
   Tem consulta gratuita? Avaliação? Qual é a porta de entrada?

Direção para reescrever:
- Gancho: deve falar de um problema real do paciente (dor, vergonha de sorrir, evita fotos)
- Corpo: 1 diferencial concreto + prova (número, especialidade, depoimento se tiver)
- CTA: específico ("Avaliação sem custo — [dia da semana] com vagas")

Aguardo versão 2.
```

## MÓDULO B — ANÁLISE DE PERFORMANCE MENSAL

### CENÁRIO B01 — Análise com Dados Reais

**Input para MIRA:**
> "Dados de performance de abril para cliente Clínica Horizonte (gestão de redes):
> - Alcance: 12.400 (era 9.800 em março, +26%)
> - Engajamento médio: 3,2% (era 2,8%)
> - Seguidores novos: +89 (era +54)
> - Post com melhor performance: reel sobre 'como funciona o clareamento' — 4.800 alcance
> - Post com pior performance: imagem de 'parabéns clínica X anos' — 340 alcance
> - Cliques no link da bio: 67 (era 44)"

**Critérios de aprovação:**
- Identifica o padrão correto: conteúdo educativo supera conteúdo institucional
- Análise do melhor post: explica POR QUÊ funcionou (não apenas lista os números)
- Análise do pior post: explica POR QUÊ não funcionou (parabéns = nenhum valor para o seguidor)
- Recomendação prática: mais educativo, menos institucional
- Identifica oportunidade de upsell: 67 cliques no link da bio = há interesse de conversão
  (oportunidade para tráfego pago)
- Formato: conciso, não mais que 1 página

**Resposta ideal (referência):**
```
ANÁLISE DE PERFORMANCE — Horizonte — Abril/2026

📈 DESTAQUES:
Alcance +26%, engajamento +0,4pp, seguidores acelerando (+65% vs. março).
Tendência positiva em todas as métricas principais.

✅ O QUE FUNCIONOU:
Reel de clareamento (4.800 alcance) → Conteúdo educativo com prática real.
O seguidor aprendeu algo e compartilhou. Regra confirmada: educar converte em alcance.

❌ O QUE NÃO FUNCIONOU:
Post de aniversário (340 alcance) → Conteúdo institucional não tem valor para o seguidor.
Padrão: posts que falam DA clínica perdem para posts que falam PARA o paciente.

📌 AJUSTE ESTRATÉGICO PARA MAIO:
Aumentar proporção de conteúdo educativo de ~30% para ~50% do calendário.
Eliminar posts institucionais puros (substituir por "bastidores" com valor).

💡 OPORTUNIDADE IDENTIFICADA:
67 cliques no link da bio = intenção de contato real. Paciente está interessado mas
não converte sozinho. Recomendar: tráfego pago para ampliar esse resultado (upsell).
Escalar para CEO-ATLAS apresentar a Yan.
```

## MÓDULO C — ESTRATÉGIA DE MARKETING DE CLIENTE

### CENÁRIO C01 — Definição de Estratégia para Novo Cliente

**Input para MIRA:**
> "Novo cliente onboardado: Advocacia Bertoldi & Associados, Joinville.
> Especialidade: direito trabalhista e empresarial. 3 advogados.
> Público-alvo: RH de empresas médias e MEIs com questões trabalhistas.
> Serviço contratado: gestão de redes sociais (Instagram + LinkedIn).

- Estabelece métricas de sucesso específicas para esse segmento (não genérico)
- Identifica o que NÃO fazer (ex: não postar jurisprudência técnica no Instagram)

**O que avaliar:** MIRA deve mostrar que entende que advocacia em SC tem cultura conservadora —
LinkedIn é mais relevante que Instagram para B2B jurídico.

## NOTAS DE CALIBRAÇÃO

**Sinal de MIRA bem calibrada:**
- Feedback para SAGE é específico ("troque X por Y"), nunca vago ("melhore")
- Análise de performance vai à causa-raiz, não apenas lista números
- Identifica oportunidade de upsell na performance (observação de FINN)
- Mantém posição contra pedido de cliente quando os dados contradizem
- Distingue o que é problema de conteúdo do que é problema de segmento

**Sinal de MIRA mal calibrada:**
- Aprova tudo que SAGE manda para "não travar"
- Culpa o algoritmo quando performance cai sem investigar
- Cede à estratégia do cliente mesmo contra os dados
- Análise de performance é só números sem interpretação

**Critério para v1.2:** 30 dias de revisões reais + análise de pelo menos 3 clientes.
