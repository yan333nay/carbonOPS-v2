# Guia de Onboarding Técnico por Serviço
**Uso:** Processo 04 — o que pedir ao cliente conforme o serviço contratado
**Responsável:** NOVA solicita → FLUX monitora prazo → CEO-ATLAS alerta se não chegar

---

> Use este guia para saber exatamente quais acessos e materiais pedir a cada
> cliente novo. Combine com o e-mail de boas-vindas (template em mensagens-padrao.md).

---

## ACESSOS UNIVERSAIS (todo cliente, sempre)

Independente do serviço contratado, sempre solicitar:

| Item | Prazo | Formato |
|------|-------|---------|
| Logotipo em alta resolução | 3 dias úteis | .ai, .pdf ou .png fundo transparente |
| Manual de marca (se houver) | 3 dias úteis | PDF |
| Paleta de cores com códigos | 3 dias úteis | Hex/RGB no mínimo |
| Tipografia utilizada | 3 dias úteis | Nome da fonte ou arquivo .ttf/.otf |
| Fotos da empresa/produto disponíveis | 5 dias úteis | Google Drive — pasta compartilhada |
| Contato do ponto focal do projeto | Imediato | Nome + WhatsApp |

## COMO MONITORAR OS ACESSOS NO CLICKUP

No checklist de onboarding de cada cliente (criado pelo FLUX):
- Cada acesso tem um item de checklist
- Quando cliente enviar o acesso: marcar como concluído + registrar em comentário
- Se prazo passar sem o acesso: FLUX alerta NOVA para fazer follow-up
- Se bloquear o início do trabalho: escalar para Yan comunicar ao cliente a consequência no prazo

**Regra de ouro:** A Carbon Films não inicia a execução sem os acessos obrigatórios.
Atraso nos acessos = atraso no início, e o cliente precisa ser comunicado.
