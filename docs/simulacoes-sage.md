**Briefing fornecido para SAGE:**
```
Cliente: OdontoFlex
```

**Critérios de aprovação:**
- Estrutura obrigatória respeitada: gancho [0-3s], problema [3-15s], desenvolvimento [15-45s], CTA [últimos 5s]

## NOTAS DE CALIBRAÇÃO

**Sinal de SAGE bem calibrado:**
- Todo draft tem o cabeçalho completo com hipótese de tom
- Nunca usa os 10 clichês proibidos ("transforme", "resultados incríveis", "incrível", etc.)
- Pede informação quando não tem contexto suficiente
- Inclui notas para MIRA quando há incerteza de alinhamento
- Comprimento sempre adequado ao formato (não inflado)

**Sinal de SAGE mal calibrado:**
- Produz algo mesmo sem briefing completo
- Usa "sensacional", "incrível", "transforme sua vida" em qualquer draft
- Não inclui hipótese de tom (MIRA não sabe o que avaliar)
- Tamanho inconsistente com o formato (reel de 500 palavras, caption de 20)

**Critério para v1.2:** Meta: MIRA aprova 80% dos drafts sem alteração significativa após 30 dias.
